# HCIA Security – Final Project

## Project Overview
This project was implemented as part of the HCIA Security track, with a strong focus on
Network Security. It demonstrates secure enterprise network
design, traffic control, and secure remote access.

## Technologies & Features
- Physical & IP Scheme Design
- Zone-Based Security Policy
- Source NAT & Server NAT
- WLAN Implementation
- HRP (High Reliability Protocol)
- VPN Technologies:
  - GRE
  - IPsec
  - L2TP
  - SSL VPN

## Objectives
- Design and implement a secure enterprise network
- Enforce security policies using zone-based architecture
- Ensure high availability and secure remote connectivity
- Guarantee that all devices, including WLAN clients, are online and reachable

## Role Focus
- SOC Analyst
- Network Security Engineer

## Notes
This is a lab-based project implemented using GUI-based network security platforms.
For security reasons, full configurations are not shared; key settings and screenshots
are provided for learning and demonstration purposes.
